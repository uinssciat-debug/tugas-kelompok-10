Page 3 — Tools Sitasi: Mendeley, Zotero, ChatGPT Citation

Manajemen referensi adalah bagian penting dari penulisan akademik. Tools sitasi membantu menyimpan sumber, mengelola PDF, menyisipkan kutipan ke dokumen, dan membuat daftar pustaka otomatis sesuai gaya (APA, MLA, Chicago, dsb). Di bawah ini uraian lengkap fitur, keunggulan, contoh alur kerja, dan tips penggunaan untuk tiga tools populer.

Mendeley — Fitur & Cara Pakai

Mendeley memungkinkan Anda menyimpan artikel jurnal (PDF), mengorganisir koleksi berdasarkan folder atau tag, dan menambahkan anotasi. Dengan plugin Mendeley Cite untuk Microsoft Word Anda dapat memasukkan sitasi langsung saat menulis. Langkah singkat:

  1. Daftar akun Mendeley (gratis) → instal aplikasi desktop atau gunakan versi web.
  2. Impor PDF atau tambahkan metadata secara manual.
  3. Pasang plugin Word (Mendeley Cite) → pilih gaya sitasi → sisipkan kutipan.

Tips: Periksa metadata (judul, penulis, tahun) setelah impor—sering perlu koreksi otomatis.

Zotero — Fitur & Cara Pakai

Zotero adalah alternatif open-source yang kuat. Kelebihannya adalah integrasi browser yang mudah untuk “menangkap” sitasi dari halaman artikel atau katalog perpustakaan. Zotero juga menyediakan penyimpanan file dan sinkronisasi. Langkah singkat:

  1. Instal Zotero dan Zotero Connector (ekstensi browser).
  2. Ketika membuka artikel, klik ikon Zotero Connector untuk menyimpan metadata & PDF.
  3. Gunakan plugin Zotero untuk Word/Google Docs untuk menyisipkan kutipan.

Tips: Gunakan folder dan tag untuk mengelompokkan referensi per bab atau topik penelitian.

ChatGPT Citation — Peran & Catatan Penting

Fitur pembuatan sitasi di ChatGPT atau model AI lain dapat mempercepat pembuatan format kutipan. ChatGPT dapat menghasilkan contoh kutipan dalam format APA/MLA/Chicago jika diberikan metadata. Namun, AI kadang membuat kesalahan pada detail (mis. halaman, DOI). Selalu verifikasi hasil di sumber primer.

Contoh penggunaan aman: minta AI membuat format kutipan lalu cek ulang di database jurnal atau perpustakaan.

Perbandingan Singkat

  • Mendeley: mudah untuk manajemen PDF & plugin Word.
  • Zotero: sangat baik untuk menangkap sitasi web & fleksibel (open-source).
  • ChatGPT Citation: cepat untuk format sitasi, tapi perlu verifikasi manual.
